# 업로드 기준

## 1차 공개 업로드

이번 저장소에는 아래 자료만 올린다.

- 직접 정리한 Markdown 문서
- 자료 분류를 설명하는 안내 문서
- 참고용 OCR 텍스트

## 제외 기준

아래 자료는 이번 공개 업로드에서 제외한다.

- 원본 PDF
- HWP, HWPX 초안 문서
- `raw_data` 전체
- 아직 정리되지 않은 개인 작업 중간본

## 이유

- 공개 저장소는 읽기 쉬운 자료가 중심이 되는 편이 좋다.
- 원본 파일은 용량이 크고 관리 포인트가 늘어난다.
- 텍스트 정리본이 있어야 저장소 목적이 더 분명해진다.
