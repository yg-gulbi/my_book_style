# my_book_style

독서토론 자료를 공개용으로 다시 정리해 올리는 저장소다.

이 저장소에는 원본 파일 전체를 그대로 올리기보다, 읽고 활용하기 좋은 정리본 중심 자료를 우선 담는다. 운영 자료, 발제 자료, 책별 정리, 토론 주제, 인용 자료를 분류해 두고 이후 필요한 범위만 천천히 확장한다.

## 이번 업로드 범위

- `정리본`: 전체 자료 분류표와 통합 요약 문서
- `분류된_자료/*/정리`: 원문을 바탕으로 다시 읽기 좋게 정리한 Markdown 자료
- `분류된_자료/01_팀장_운영/통합정리`: 운영 철학과 진행 기준을 묶은 통합 문서
- `분류된_자료/README.md`: 분류 체계 설명

## 이번에 제외한 자료

- `PDF`: 원본 PDF
- `raw_data`: 원본 보관 폴더
- `*.hwp`, `*.hwpx`: 초안 및 문서 원본

이 자료들은 저장소 크기를 불필요하게 키우거나, 공개 저장소 기준으로 다루기 조심스러운 원본성 자료일 수 있어 이번 1차 업로드에서는 제외했다. 텍스트 추출본 같은 raw 성격의 산출물도 이 저장소에서는 다루지 않는다.

## 폴더 구조

```text
my_book_style
├─ 분류된_자료
│  ├─ 01_팀장_운영
│  ├─ 02_발제문_토론질문
│  ├─ 03_책내용_개인정리
│  ├─ 04_토론주제
│  └─ 05_명언_인용
└─ 정리본
```

## 활용 방향

- 팀 운영 기준은 `01_팀장_운영`과 `정리본`에서 먼저 본다.
- 실제 토론에 바로 쓰려면 `02_발제문_토론질문`과 `04_토론주제`를 본다.
- 책별 생각 정리와 토론 확장은 `03_책내용_개인정리`를 중심으로 본다.
- 도입 문장이나 마무리 인용은 `05_명언_인용`을 활용한다.
