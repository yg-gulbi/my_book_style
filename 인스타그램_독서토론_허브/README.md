# 인스타그램 독서토론 허브

`책의 아고라`는 인스타그램을 기반으로 책을 좋아하는 사람들이 질문하고, 반론하고, 서로의 생각을 더 정확하게 만드는 독서토론 채널이다.

이 폴더는 계정을 바로 운영할 수 있도록 만든 실행 문서 모음이다. 단순한 계획이 아니라 프로필 문구, 고정 게시물 초안, 첫 4주 콘텐츠, 댓글 운영 기준, 성과 점검표까지 한 번에 볼 수 있게 구성했다.

## 운영 핵심

- 타깃: 책 좋아하는 일반인
- 핵심 형식: 발제 카드뉴스
- 업로드 리듬: 주 2회
- 1차 목표: 댓글 토론 만들기
- 확장 방향: 온라인 댓글 토론일, 이후 오프라인 독서토론 연결

## 문서 구성

- `01_채널_운영전략.md`: 채널 정체성, 프로필, 콘텐츠 축, 성장 단계
- `02_고정게시물_3종_초안.md`: 채널 소개, 좋은 반론 규칙, 첫 토론 참여 방법
- `03_4주_콘텐츠_캘린더.md`: 첫 4주 동안 올릴 8개 게시물 구성
- `04_카드뉴스_템플릿.md`: 6장 카드뉴스 제작 템플릿
- `05_댓글_운영_가이드.md`: 댓글 문화, 반론 기준, 숨김/삭제 기준
- `06_성과_체크리스트.md`: 첫 8개 게시물과 월별 성장 점검표

## 반복 문구

```text
좋은 반론 환영합니다.
다만 반론에는 이유, 경험, 책 내용 중 하나를 붙여주세요.
사람을 공격하지 말고 생각을 밀어주세요.
```
