# 분류된 자료 안내

이 폴더는 원본 PDF를 옮긴 것이 아니라, 보기 쉽게 분류별로 복사해둔 자료입니다.

각 분류 폴더 안에는 다음처럼 자료가 카테고리 단위로 모여 있습니다.

- `PDF`: 해당 분류의 원본 PDF 모음
- `텍스트추출`: PDF에서 OCR로 뽑은 원문 텍스트 모음
- `정리`: OCR 원문을 바탕으로 읽기 좋게 다시 묶은 정리본 모음
- `01_팀장_운영` 폴더에는 여러 자료를 한데 묶은 `통합정리` 폴더도 따로 있습니다.

## 분류 기준

- `01_팀장_운영`: 독서토론 팀장 역할, 모임 운영 방식, 첫 모임 기획, 팀원 이해 자료
- `02_발제문_토론질문`: 실제 토론에서 사용할 발제문과 질문 자료
- `03_책내용_개인정리`: 책 내용 요약, 개인 생각, 책 배경 정리
- `04_토론주제`: 책과 별개로 사용할 수 있는 찬반/가치 판단형 토론 자료
- `05_명언_인용`: 인상 깊은 문장, 토론 도입이나 마무리에 쓸 수 있는 인용 자료

자세한 전체 분류 해설은 `정리본` 폴더의 문서를 보면 됩니다.
